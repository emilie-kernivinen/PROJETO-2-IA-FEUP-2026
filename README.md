## Project Overview
This project aims to predict whether a **Spotify Premium user is likely to cancel their subscription within the next 30 days**.
The project uses synthetically generated user data (5000 users) designed to simulate realistic listening behavior and subscription activity patterns.


## How to Run the Project

Inside the `app` folder, run:

```bash
python -m streamlit run app.py
```
